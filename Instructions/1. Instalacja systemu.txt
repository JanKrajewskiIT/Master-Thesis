1. Został zainstalowany system CentOS 7 Linux ze środowiskiem gnome.

2. Aby móc logować się na roota poprzez sudo su został skonfigurowany plik sudoers :
su -
gedit /etc/sudoers

Gdzie dodano odpowiednią linię w sekcji:
## Allow root to run any commands anywhere 
magisterka ALL=(ALL) ALL

3. Kolejną czynnością było zainstalowanie pluginu do VB guest addition
rpm -Uvh http://dl.fedoraproject.org/pub/epel/7/x86_64/e/epel-release-7-5.noarch.rpm;
yum -y install dkms kernel-devel-$(uname -r) kernel-headers-$(uname -r);
wget http://download.virtualbox.org/virtualbox/5.0.0/VBoxGuestAdditions_5.0.0.iso;
mount -o loop VBoxGuestAdditions_5.0.0.iso /media/;
/media/VBoxLinuxAdditions.run;

4. Do połączenia z internetem mamy interface enp0s3, z którego będziemy korzystać w kolejnych zadaniach.

5. Dalej restartujemy system i zapisujemy snapshoot maszyny virtualnej.
