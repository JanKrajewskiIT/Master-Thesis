1. Najpierw instalujemy postgresql.
yum install postgresql postgresql-devel postgresql-server postgresql-contrib postgresql-libs postgresql-client
systemctl enable postgresql
postgresql-setup initdb
systemctl status postgresql.service
systemctl start postgresql

2. Do uzyskania zamierzone działania barnyard2 potrzebne nam będzie wcześniejsze skonfigurowanie środowiska bazy danych.
sudo su
su - postgres
psql

create database snort_logs;
create user snort_user PASSWORD 'magisterka';
grant ALL ON DATABASE snort_logs to snort_user;
systemctl reload postgresql

3. Drugim krokiem jest odpowiednie skonfigurowanie pliku pg_hba.conf
gedit /var/lib/pgsql/data/pg_hba.conf

# IPv4 local connections:
#host    all             all             127.0.0.1/32            ident
host    snort_logs      snort_user   	 127.0.0.1/32    	 password

4. Następnie należy zalogować się do przygotowanej bazy danych stworzonymi kredkami.
systemctl reload postgresql
psql -U snort_user -h 127.0.0.1 snort_logs

5. Kolejnym krokiem jest wrzucenie schematu barnyard2
\i /home/magisterka/Pulpit/Magisterka/pluginy/barnyard2-stable/schemas/create_postgresql
